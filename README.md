# Reflejos - Juego de tiempo de reacción

Este proyecto es un juego interactivo desarrollado con **React** que mide el **tiempo de reacción** del jugador frente a un estímulo visual (una bolita). El jugador debe hacer clic lo más rápido posible una vez que la bolita aparece, y su tiempo será registrado y comparado con los mejores puntajes.

---

## 🚀 Características principales

- Ingreso de nombre del jugador.
- Estímulo visual aleatorio (bolita).
- Medición del tiempo de reacción en milisegundos.
- Registro del puntaje del jugador en una base de datos **Supabase**.
- Visualización de los 5 mejores puntajes.
- Estilo responsivo con **Tailwind CSS**.

---

## 🛠️ Tecnologías utilizadas

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Supabase](https://supabase.com/) 
- [Tailwind CSS](https://tailwindcss.com/)

## 🔧 Instalación y ejecución local

Sigue estos pasos para clonar y correr el proyecto en tu máquina:

```bash
# 1. Clona el repositorio
git clone https://github.com/Robin0980robin/reflejos.git

# 2. Accede al directorio del proyecto
cd reflejos

# 3. Instala las dependencias
npm install

# 4. Crea un archivo .env con tu clave pública de Supabase
VITE_SUPABASE_KEY=TU_CLAVE_PUBLICA

# 5. Ejecuta la app
npm run dev
